.Project: QR Code Generator
Author: Sandeep271
Description: This Python script generates a QR code using the 'qrcode' and 'Pillow' libraries. 
It takes a URL or any text input and creates a PNG image file with the corresponding QR code.

Dependencies:
- Python 3.x
- qrcode
- Pillow

Files Included:
- generate_qr.py (Python source code)
- my_qrcode.png (Output QR code image)
- manifest.txt (This file)

Instructions to Run:
1. Ensure Python and required libraries are installed.
2. Open terminal in the project directory.
3. Run the script: python generate_qr.py
4. The QR code image will be saved as 'my_qrcode.png'.

Output:
- my_qrcode.png
